# Firebase API Console App (.NET)

This project is a simple .NET Console application that interacts with Firebase Realtime Database using HTTP requests.  
It demonstrates how to perform POST operations and send JSON data using a custom ApiHandler class.  
The application uses HttpClient with async/await for non-blocking API calls.  
Data is sent to Firebase and responses are handled and displayed in the console.  
This project helps understand REST APIs, HTTP methods, and Firebase integration in .NET.
